# vuepress 搭建博客

我基于 vuepress 2.0在新窗口打开 搭建了静态博客，作为人民网络的官网。

代码库地址 github.com/rmw-link/blog-vuepress2在新窗口打开

定制的内容如下 :

  • 利用 md-padding在新窗口打开 和 git 钩子在新窗口打开 ，自动给中文和英文之间加上空格在新窗口打开

    git clone 之后，进入目录，输入 direnv allow，会执行 .envrc在新窗口打开 自动安装 git 钩子。

    direnv在新窗口打开 是环境自动加载工具。点此查看教程在新窗口打开。

  • 自动翻译博客为多国语言在新窗口打开

    参见 markdown 翻译工具

  • 改名主语言文件名后，自动清理其他语言的旧文件在新窗口打开

  • 翻译主题配置的脚本在新窗口打开

  • 监控改动并编译 stylus 样式和 coffeescript 脚本到 site/.vuepress在新窗口打开

    此处用到小工具 concurrently在新窗口打开 支持并发运行多个命令行任务，参数 --kill-others 可以保证同一时间只有一个脚本运行

  • 修改中文字体为 Noto Sans SC在新窗口打开

  • 批量下载 css 文件中字体的脚本在新窗口打开

    基于 xonsh在新窗口打开 编写，./download.xsh code.css 这样就可以下载 css 文件中的字体并生成对应的 stylus。

    xonsh 融合了 python 和 bash，非常实用。

  • 访问首页会自动切换浏览器的默认语言在新窗口打开

  • 扫描目录自动映射为网址在新窗口打开

  • sitemap.xml 生成在新窗口打开
