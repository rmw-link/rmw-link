# Mi biblioteca de herramientas node.js

  • @rmw/axiosAbrir en una nueva ventana axios con tiempos de espera y reintentos

  • @rmw/csvAbrir en una nueva ventana lee el CSV línea por línea

  • @rmw/md-includeAbrir en una nueva ventana hace que markdown soporte #include /xx/xx.xxxpara referenciar el código fuente al escribir readme.mdAbrir en una nueva ventana, ver coffee_template/git/hooks/pre-commitAbrir en una nueva ventana

  • @rmw/poolAbrir en una nueva ventana Promete piscinas concurrentes, recuerda poner bind bajo las funciones utilizadas para class

  • @rmw/qiniuAbrir en una nueva ventana Subiendo directorios a Seven Bulls CloudAbrir en una nueva ventana

  • @rmw/source-map-support Que source-map-support soporte la visualización en chinoAbrir en una nueva ventana

  • @rmw/walkAbrir en una nueva ventana itera a través de las rutas de los directorios de forma asíncrona, donde import {walkRel} from '@rmw/walk' es una iteración de ruta relativa

  • @rmw/thisdirAbrir en una nueva ventana pasa a import.metapara devolver la ruta del directorio del archivo actual; import {thisfile} from '@rmw/thisdir' devuelve la ruta del propio archivo

  • @rmw/consoleAbrir en una nueva ventana hace que los números de línea de salida de la consola sean los siguientes

    10:45:18 coffee-template/src/index.coffee:11 › 1

  • coffee_templateAbrir en una nueva ventana coffeescript project template : node.js' --experimental-loader soporta el tipo: módulo, es decir, la importación al estilo ES6
