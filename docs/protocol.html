# 1 · 人民网络通讯协议设计

在构建软件之前，先要定义通讯协议。

我们的目标是实现无中央服务器的对等网络 ：在这个架构中，人人平等，没有人垄断数据。

# 身份

加入网之前的第一件事是就是生成身份标识。

身份标识是Ed25519密钥对。

首次启动时，客户端会自动生成一个Ed25519密钥对并将其保存在数据目录下的 ed25519.pk 和 ed25519.sk 。

不同设备可以使用同一组秘钥，也就是说，你的平板、手机、电脑可以同时在线。
