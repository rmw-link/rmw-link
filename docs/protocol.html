# 1 · 人民网络通讯协议设计

在构建软件之前，先要定义通讯协议。

我们的目标是实现无中央服务器的对等网络 ：在这个架构中，人人平等，没有人垄断数据。

# 身份秘钥

加入人民网络前，首先要生成身份秘钥。

身份秘钥是Ed25519密钥对。

首次启动时，客户端会自动生成一个密钥对。

不同设备可以使用同一组秘钥，也就是说，你的平板、手机、电脑可以同时在线。

之后，用户可以选择给自己起一个昵称。昵称可以修改，但身份秘钥永远不变。

如果身份秘钥丢失了，只能放弃原有的账户，重新生成一个新的身份。
