# 私のnode.jsツールライブラリ

  • タイムアウトとリトライを備えた@rmw/axios新しいウィンドウで開く axios

  • rmw/csv新しいウィンドウで開く は CSV を一行ずつ読み込む。

  • @rmw/md-include新しいウィンドウで開く は readme.md新しいウィンドウで開く を書くときにソースコードを参照するための markdown サポート #include /xx/xx.xxxを作る。 coffee_template/git/hooks/pre-commit新しいウィンドウで開くを参照。

  • @rmw/pool新しいウィンドウで開く Promise の同時使用プールで、 class に使用する関数の下に bind を置くことを忘れないように。

  • @rmw/qiniu新しいウィンドウで開く ディレクトリを Seven Bulls Cloudに新しいウィンドウで開くアップロードする。

  • @rmw/source-map-support source-map-support で中国語表示をサポートするように新しいウィンドウで開くした。

  • @rmw/walk新しいウィンドウで開く はディレクトリのパスを非同期で反復します。 import {walkRel} from '@rmw/walk' は相対パスの反復です。

  • @rmw/thisdir新しいウィンドウで開く は import.metaを渡して，現在のファイルディレクトリへのパスを返す。import {thisfile} from '@rmw/thisdir' はファイル自体へのパスを返す。

  • @rmw/console新しいウィンドウで開く は、コンソール出力の行番号を（以下のように）変更させます。

    10:45:18 coffee-template/src/index.coffee:11 › 1

  • coffee_template新しいウィンドウで開く coffeescript project template : node.js' --experimental-loader supported type: module, すなわち ES6-style import
