# Моя библиотека инструментов node.js

  • @rmw/axiosОткрыть в новом окне axios с таймаутами и повторными попытками

  • @rmw/csvОткрыть в новом окне читает CSV построчно

  • @rmw/md-includeОткрыть в новом окне делает поддержку markdown #include /xx/xx.xxxдля ссылок на исходный код при написании readme.mdОткрыть в новом окне, см. coffee_template/git/hooks/pre-commitОткрыть в новом окне

  • @rmw/poolОткрыть в новом окне Обещайте одновременные пулы, не забудьте поместить bind под функции, используемые для class.

  • @rmw/qiniuОткрыть в новом окне Загрузка каталогов в Seven Bulls CloudОткрыть в новом окне

  • @rmw/source-map-support Пусть source-map-support поддерживает отображение на китайском языкеОткрыть в новом окне

  • @rmw/walkОткрыть в новом окне асинхронно перебирает пути каталогов, где import {walkRel} from '@rmw/walk' - итерация по относительному пути.

  • @rmw/thisdirОткрыть в новом окне передает import.meta, чтобы вернуть путь к текущему каталогу; import {thisfile} from '@rmw/thisdir' возвращает путь к самому файлу.

  • @rmw/consoleОткрыть в новом окне заставляет консоль выводить номера строк (следующим образом)

    10:45:18 coffee-template/src/index.coffee:11 › 1
